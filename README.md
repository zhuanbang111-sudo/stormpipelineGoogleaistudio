# StormFlow Designer - 城市雨水管网水力设计与模拟系统

StormFlow Designer 是一款专为城市雨水管网设计的在线水力建模与分析工具。它集成了地理信息系统 (GIS)、水力计算引擎以及标准化的工程报告输出功能，旨在为排水工程师提供一个高效、直观的设计平台。

## 核心功能

### 1. 交互式管网建模
*   **节点与管线绘制**：支持在地图上直接添加检查井 (Manhole)、排放口 (Outfall) 并连接管线 (Pipe)。
*   **汇水区 (Catchment) 划定**：支持多边形绘制汇水面积，并可关联至特定的排放节点。
*   **智能吸附**：在绘制管线时自动吸附至最近节点，确保拓扑结构准确。

### 2. 精准水力计算引擎
*   **推理公式法 (Rational Method)**：严格遵循《室外排水设计标准》GB50014-2021。
*   **深圳专项支持**：预设深圳西部、中部、东部三区的暴雨强度公式，支持自定义公式参数。
*   **动态历时计算**：支持各汇流面独立设置地面集水时间 ($t_1$)，系统自动计算累计降雨历时 ($t$)。
*   **管径智能推荐**：根据计算流量和流速限制，自动推荐标准管径并计算优化后的水力状态。

### 3. GIS 与底图集成
*   **国产底图优化**：默认采用**天地图 (Tianditu) WGS84** 矢量/影像底图，确保国内访问顺畅且注记清晰。
*   **坐标转换**：内置坐标转换引擎，支持将 WGS84 经纬度导出为 **CGCS2000** 投影坐标系。
*   **底图导入**：支持导入 CAD/GIS 导出的背景要素，辅助管网布设。

### 4. 成果导出与报告
*   **DXF 工程图纸**：一键导出包含节点、管线、汇水区图层的 DXF 文件，可直接在 AutoCAD 中打开。
*   **水力分析报告**：生成包含节点、管线、汇水区详细水力参数的 CSV 报告，满足工程评审需求。

## 实现逻辑

### 技术栈
*   **前端框架**：React 18 + Vite
*   **地图引擎**：Leaflet (react-leaflet)
*   **状态管理**：自定义 Hook (`useNetworkStore`)，支持完整的撤销/重做 (Undo/Redo) 逻辑。
*   **样式处理**：Tailwind CSS
*   **数学计算**：
    *   **球面距离**：Haversine 公式计算管段物理长度。
    *   **水力学**：曼宁公式 (Manning's Equation) 计算过流能力与流速。
    *   **坐标转换**：Proj4js 实现经纬度到投影坐标的转换。

### 计算流程
1.  **拓扑分析**：系统自动对管网进行拓扑排序，确定从上游到下游的计算顺序。
2.  **流量累加**：根据汇水区面积 ($A$)、径流系数 ($\psi$) 和降雨强度 ($q$) 逐级累加流量。
3.  **历时迭代**：$t = t_1 + m \cdot t_2$，其中 $t_2$ 根据管内流速动态更新。
4.  **状态判定**：对比计算流量与管段能力，判定满管 (Surcharge) 或溢流 (Flood) 状态。

## 使用方法

### 第一步：环境设置
1.  点击顶部 **Settings**，选择合适的底图类型（建议使用天地图）。
2.  在右侧 **Hydraulic Engine** 面板选择计算区域（如深圳西部）和重现期。

### 第二步：管网布设
1.  使用底部工具栏的 **Add Manhole** 在地图上布置检查井。
2.  使用 **Add Pipe** 连接节点，注意坡度由节点标高差决定。
3.  使用 **Add Catchment** 绘制汇水范围，并在属性栏中选择对应的**地面种类**（系统将自动推荐径流系数）。

### 第三步：运行模拟
1.  点击右侧面板底部的 **Run Simulation**。
2.  观察地图：红色管段表示满管，红色闪烁节点表示溢流，蓝色表示正常过流。

### 第四步：查看与导出结果
1.  点击底部 **Simulation Results** 面板，查看流量过程线和详细数据表。
2.  在表格中查看**推荐管径**和**推荐流速**。
3.  使用顶部 **Export DXF** 导出图纸，或点击 **Reports** 下载完整计算书。
